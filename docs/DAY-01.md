# Day 1 – Project Setup

## Overview

The first day focused on setting up the development environment and creating the initial project structure for Scanly. The frontend and backend were initialized independently to provide a solid foundation for future development.

## Completed Tasks

* Initialized the project repository.
* Created the React (Vite) frontend.
* Created the FastAPI backend.
* Configured CORS for frontend-backend communication.
* Added environment variable support using `.env`.
* Created health check endpoints.
* Organized the project directory structure.

## Technologies Used

* React (Vite)
* FastAPI
* Python
* Node.js
* Uvicorn
* python-dotenv

## Project Structure

```text
scanly/
├── scanly-backend/
└── scanly-frontend/
```

## Outcome

* Backend server running successfully.
* Frontend running successfully.
* API documentation available through Swagger UI.
